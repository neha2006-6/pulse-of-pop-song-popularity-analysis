#  Pulse of Pop: Song Popularity Analysis

##  Overview
This project analyzes music data to understand the factors that influence song popularity. It combines data analysis and machine learning techniques to predict and classify songs as Hit or Flop.

##  Objectives
- Predict song popularity using machine learning models  
- Classify songs into Hit or Flop categories  
- Analyze the impact of audio features on song success  

##  Machine Learning Models Used
- Linear Regression (for prediction and threshold-based classification)  
- Random Forest (for classification and improved accuracy)  

##  Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Google Colab  

##  Methodology
- Data preprocessing and cleaning  
- Feature selection and analysis  
- Model training using Linear Regression and Random Forest  
- Classification of songs into Hit or Flop  
- Model comparison  

##  Sample Output

| Case     | Linear Regression | Random Forest |
|----------|------------------|---------------|
| Sample 1 | Hit              | Hit           |
| Sample 2 | Flop             | Flop          |
| Sample 3 | Flop             | Hit           |

##  Results
- Both models successfully classified songs based on popularity  
- Random Forest performed better for classification tasks  
- Differences in predictions highlight model behavior  

##  Conclusion
This project demonstrates how machine learning can be used to analyze and predict song popularity. It provides useful insights into what makes a song successful.
